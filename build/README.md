# Build Assets

Place your application icon here:

- **icon.ico** - Windows icon file (256x256 recommended)

You can create an icon from an image using online tools like:
- https://www.icoconverter.com/
- https://convertico.com/

## Recommended Icon Sizes:
- 256x256
- 128x128
- 64x64
- 48x48
- 32x32
- 16x16

All sizes should be included in the .ico file.
